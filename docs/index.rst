pysvd2dts is used to create an initial zephyr devicetree file from a svd file.



CMSIS System View Description format (SVD)
##########################################

CMSIS System View Description format (SVD) is a formal description of the 
an Arm Cortex-M processor-based microcontrollers memory mapped peripherals registers.

`<https://www.keil.com/pack/doc/CMSIS/SVD/html/svd_format_pg.html>`


Devicetree (DTS)
################

A devicetree (DTS) is primarily a hierarchical data structure that describes hardware. 

`<https://docs.zephyrproject.org/latest/build/dts/index.html>`_
